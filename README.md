## What This Is

A live, interactive implementation of the **LOGOS Five-Layer Governance Stack** as defined in LOGOS-STD-GOV-001. Submit any AI agent action or prompt and watch it evaluated in real time across all five governance layers:

| Layer | Component | Function |
|---|---|---|
| L-1 | Constitutional Bounds | Absolute prohibitions — no override possible |
| L0 | Intent Verification | Cryptographic intent alignment check |
| L1 | Sentinel Gates | Pattern-based single-turn threat detection |
| L2 | OverWatch | Multi-turn drift and trajectory analysis |
| L3 | Execution Interlock | Dynamic Risk Score (DRS) routing decision |

## Red-Team It

This sandbox is built to be probed. Try to bypass it. Every interaction is evidence — for the standard's CI/CD loop, for foresight updates, and for public accountability.

## Access Model

- **Interact-only.** The governance engine system prompt and layer logic are not exposed.
- **Non-forkable IP.** The standard is public. The implementation architecture is proprietary.
- **Bring your own API key.** Your key stays in your browser. Nothing is sent to LOGOS servers.
- **All interactions are logged immutably** to the LOGOS evidence registry via GitHub Actions.
